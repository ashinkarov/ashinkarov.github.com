Tool for user interface generation

Šinkarovs, Artjoms

The Bachelor Paper under consideration deals with the problem of user interface creation and support. The paper studies the approach, which helps to create a user’s interface configuration and transform it into various kinds of user’s interfaces.

The paper uses a specially developed library written in Python to create a user’s interface. The library has been made so that it can easily be adapted and enriched. The method can also be used for devices other than personal computer, as well as for programs with a different architecture type.
